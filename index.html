PikaDish

Your Weekly Meal Planner

PikaDish is a personal project that I undertook with 3 other collegues. It is a mobile and web application that allows you to plan every meal you wish to cook in the given week, and then tally's a list of the required ingredients to cook these meals. As you can imagine, this is very helpful when planning your weekly shop.

You can view my website here.

Previous Next


To create this application, we used a Asp.Net Web API with a SQL database in the backend, and Angular JS, JQuery and Bootstrap in the frontend.

Previous Next


To fully utilize the potential of this application, I set about converting it to a mobile application aswell. The benefit of this was ease of use for the consumer, so that when they were away from a laptop,they could easily access their shopping list on their mobile device.

To create the mobile version, I used a mobile development framework called PhoneGap..
